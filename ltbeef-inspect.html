How to disable extensions on the latest version if you have inspect allowed:
chrome-extension://gndmhdcefbhlchkhipcnnbkcmicncehk/manifest.json

^ Go to this URL on a school ChromeOS account
Open inspect
Click console

Run basic LTBEEF code such as
chrome.management.setEnabled('extensionid',!1)


To turn back on an extension (idk why you would), use this code
chrome.management.setEnabled('extensionid',!0)


(Replacing extensionid with the ID of the extension you want to disable, e.g. the stuff after the equals sign in the URL bar when you click its "details" button in chrome://extensions)
